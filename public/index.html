Уведомления, когда я в эфире

Человек открывает ссылку один раз, включает уведомления, а дальше получает push, когда ты выходишь в прямой эфир.

Для iPhone нужно добавить сайт на экран «Домой» и открыть его оттуда. Для Android обычно достаточно разрешить уведомления, а установка на экран — по желанию. citeturn759452search0turn759452search1turn759452search4

Для участников

Нажми кнопку ниже и разреши уведомления. Если ты на iPhone, сначала установи сайт на экран «Домой».

iPhone: перед включением уведомлений
  1. Открой ссылку в Safari
  2. Нажми «Поделиться»
  3. Выбери «На экран Домой»
  4. Открой приложение с экрана Домой
  5. Нажми «Включить уведомления»
Android
  1. Открой ссылку в Chrome
  2. Нажми «Включить уведомления»
  3. При желании установи сайт на экран через меню браузера
Пока не подключено.

Если кнопка установки не показывается, открой сайт в Safari или Chrome и используй стандартное меню браузера.

Для автора

Когда запускаешь эфир, отправь уведомление всем подписчикам. Можно использовать ссылку по умолчанию или вставить новую прямо перед рассылкой.
